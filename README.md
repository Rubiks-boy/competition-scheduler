# Competition Scheduler

Competition Scheduler is a React + TypeScript app for generating schedules for official World Cube Association competitions.

## Development

Install dependencies:

```bash
yarn install
```

Start the Vite dev server:

```bash
yarn start
```

The app is available at the local URL printed by Vite (typically `http://localhost:5173`).

## Testing

Run tests in watch mode:

```bash
yarn test
```

Run tests once:

```bash
yarn test:run
```

## Type Checking

```bash
yarn typecheck
```

## Production Build

Build production assets:

```bash
yarn build
```

Preview the production build locally:

```bash
yarn preview
```
